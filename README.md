# Customer Segmentation & Campaign Response Prediction

## Project Overview
This project analyzes FMCG customer data to understand purchasing behavior, segment customers, and predict marketing campaign responses.

The objective is to help businesses identify high-value customers and optimize marketing strategies.

## Dataset
The dataset contains approximately 2240 customers and 24 variables including:

- Demographics
- Purchase behavior
- Product spending
- Marketing campaign response
- Channel usage (web, store, catalog)

## Methods Used
The project combines customer segmentation and predictive modeling:

• RFM Analysis  
• K-Means Clustering  
• Gaussian Mixture Model (GMM)  
• PCA Visualization  
• XGBoost Campaign Response Prediction

## Key Insights
- High-value customers prefer catalog and in-store purchases.
- Customers with higher spending have a higher campaign response rate.
- Previous campaign acceptance is a strong predictor of future responses.

## Business Impact
This analysis helps marketing teams:

- Identify high-value customer segments
- Target campaigns more effectively
- Improve conversion rates
- Reduce marketing costs
